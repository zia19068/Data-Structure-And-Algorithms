# Data Structures & Algorithms (DSA)

A clean, organized repository of my implementations of fundamental data structures and solutions to competitive programming problems.

## 📂 Repository Structure

- `Data-Structures`: Implementations of Linked Lists, Trees, Graphs, Heaps, etc.
- `Algorithms`: Sorting, Searching, Dynamic Programming, and Graph algorithms.
- `Problem-Solutions`: My solutions to curated problems, categorized by difficulty.

## 🛠️ Languages Used
* **Primary:**  C++ 
* **Testing:** Dev C++

## 🚀 Key Implementations

### Data Structures
- **Linear:** Stacks, Queues, Singly/Doubly Linked Lists.
- **Non-Linear:** Binary Search Trees, AVL Trees, Tries.
- **Graphs:** Adjacency List representations.

### Algorithms
- **Sorting:** Merge Sort, Quick Sort, Heap Sort.
- **Search:** Binary Search, BFS, DFS.
