
# 🚀 Data Structures & Algorithms (C++)

Welcome to the comprehensive repository of core Computer Science foundations. This project focuses on high-performance C++ implementations of essential data structures and algorithms without relying on the Standard Template Library (STL), providing a deep understanding of memory management and pointer logic.

---

## 🛠 Core OOP Concepts
Before diving into structures, this repo masters the C++ object lifecycle:
- **Abstract Data Types (ADTs):** Pure interface design.
- **Dynamic Memory:** Explicit heap allocation using `new` and `delete`.
- **The Rule of Three:** 1. **Destructor**: Memory cleanup.
  2. **Copy Constructor**: Deep copying logic.
  3. **Assignment Operator**: Safe resource transfer.

## 📦 Linear Data Structures
### Lists
- **Doubly Linked List:** Efficient bidirectional traversal.
- **Circular Linked List:** End-to-start node linking.

### Stacks & Queues
- **Stacks:** LIFO implementation (Array & Linked List).
- **Queues:** FIFO implementation.
- **Circular Queue:** Optimized array usage using modulo arithmetic.

## 🌳 Non-Linear Data Structures
- **Binary Search Tree (BST):** - Dynamic node insertion and deletion.
  - **Traversals:** Pre-order, In-order (Sorted), and Post-order.

## ⚡ Algorithms
### Sorting
| Algorithm | Logic | Complexity |
| :--- | :--- | :--- |
| **Quick Sort** | Pivot Partitioning | O(n log n) |
| **Merge Sort** | Divide & Conquer | O(n log n) |
| **Comb Sort** | Gap-based Bubble | Optimized O(n²) |
| **Bubble/Selection/Insertion** | Comparison Based | O(n²) |

### Searching
- **Linear Search:** Simple scan.
- **Binary Search:** Logarithmic search for sorted data.
- **Interpolation Search:** Formula-based estimation for uniform distributions.

## 🔄 Expression Conversion
- **Infix to Postfix:** Operator precedence handling.
- **Postfix to Infix:** Expression rebuilding with parentheses.
